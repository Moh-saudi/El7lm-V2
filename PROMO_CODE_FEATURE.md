## ✅ **ما تم إضافته:**

### **1. Admin Panel - حقل الرمز:**
```typescript
<label>
  رمز الخصم (Promo Code)
  <span>• اختياري - إذا تركته فارغاً، سيطبق تلقائياً</span>
</label>
<input
  value={formData.code || ''}
  onChange={(e) => onChange({ ...formData, code: e.target.value.toUpperCase() })}
  placeholder="SUMMER2024"
  maxLength={20}
  className="font-mono"
/>
```

### **2. صفحة الدفع - حقل إدخال:**
```typescript
<input
  type="text"
  value={promoCode}
  onChange={(e) => setPromoCode(e.target.value.toUpperCase())}
  placeholder="أدخل الرمز الترويجي"
  className="font-mono text-lg uppercase"
/>
<button onClick={handleApplyPromoCode}>
  تطبيق
</button>
```

### **3. دالة التحقق والتطبيق:**
```typescript
const handleApplyPromoCode = () => {
  // البحث عن العرض بالرمز
  const offerWithCode = availableOffers.find(
    offer => offer.code && offer.code.toUpperCase() === promoCode.toUpperCase()
  );

  if (!offerWithCode) {
    setPromoCodeError('رمز الخصم غير صحيح');
    return;
  }

  // تطبيق العرض
  setAppliedOffer(offerWithCode);
  setPromoCodeSuccess(true);
};
```
