## 💡 كيفية إضافة أسعار مخصصة جديدة

### الطريقة البسيطة (عبر ملف التهيئة):

1. افتح: `src/lib/services/init-real-pricing.ts` أو `src/lib/pricing/pricing-service.ts`
2. أضف في `DEFAULT_PLANS` → `overrides`:

```typescript
overrides: {
    'EG': {
        currency: 'EGP',
        original_price: 200,
        price: 120,
        active: true
    },
    'SA': {
        currency: 'SAR',
        original_price: 150,
        price: 90,
        active: true
    },
    'AE': {
        currency: 'AED',
        original_price: 140,
        price: 85,
        active: true
    }
}
```

3. احفظ
4. افتح: `/dashboard/admin/init-pricing`
5. اضغط "بدء التهيئة"
6. **تم!** ✅ الأسعار الجديدة ستظهر تلقائياً في:
   - تبويب "الأسعار المخصصة" (للأدمن)
   - صفحة البلك بايمنت (للعملاء)
