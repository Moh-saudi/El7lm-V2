# تحسينات صفحة إدارة المستخدمين

## الميزات المضافة

تم تحسين صفحة إدارة المستخدمين `/dashboard/admin/users` بإضافة الوظائف التالية:

### 1. البحث برقم الهاتف ✅
- البحث بالاسم والبريد الإلكتروني ورقم الهاتف في حقل البحث الموحد
- يعمل مع جميع أرقام الهواتف بمختلف التنسيقات

### 2. إيقاف الحسابات مؤقتاً ⏸️
- زر "إيقاف مؤقت" لإيقاف أي حساب مؤقتاً
- إدخال سبب الإيقاف إلزامي
- يتم حفظ:
  - `isActive: false`
  - `suspendReason`: سبب الإيقاف
  - `suspendedAt`: تاريخ الإيقاف
  - `suspendedBy`: معرف المسؤول الذي أوقف الحساب

### 3. تفعيل الحسابات الموقوفة ✅
- زر "تفعيل الحساب" للحسابات الموقوفة
- عرض سبب الإيقاف السابق عند التفعيل
- يتم حفظ:
  - `isActive: true`
  - `reactivatedAt`: تاريخ إعادة التفعيل
  - `reactivatedBy`: معرف المسؤول الذي فعّل الحساب

### 4. حذف الحسابات 🗑️
- زر "حذف الحساب" لحذف الحسابات
- الحذف منطقي (Soft Delete) - يضع علامة فقط
- يمكن للمستخدم التسجيل مرة أخرى بنفس البريد
- يتم حفظ:
  - `isDeleted: true`
  - `deletedAt`: تاريخ الحذف
  - `deletedBy`: معرف المسؤول الذي حذف الحساب

### 5. تغيير نوع الحساب 🔄
- زر "تغيير النوع" لتغيير نوع حساب المستخدم
- متاح للإدارة فقط
- يمكن التغيير بين:
  - مستخدم عادي
  - لاعب
  - أكاديمية
  - نادي
  - وكيل
  - مدرب
- يتم حفظ:
  - `accountType`: النوع الجديد
  - `accountTypeChangedAt`: تاريخ التغيير
  - `accountTypeChangedBy`: معرف المسؤول
  - `previousAccountType`: النوع السابق

### 6. توثيق ورفض الحسابات ✓✗
- زر "توثيق" للحسابات في حالة `pending`
- زر "رفض التوثيق" للحسابات في حالة `pending`
- يظهران فقط للحسابات غير الموثقة
- يتم حفظ:
  - `verificationStatus`: verified أو rejected
  - `verifiedAt` / `rejectedAt`: تاريخ الإجراء
  - `verifiedBy` / `rejectedBy`: معرف المسؤول

### 7. رسائل تسجيل الدخول 📩
- عند محاولة تسجيل الدخول من حساب موقوف، تظهر رسالة:
  ```
  تم إيقاف حسابك مؤقتاً.

  السبب: [سبب الإيقاف]

  يرجى التواصل مع الإدارة لإعادة تفعيل الحساب.
  ```
- عند محاولة تسجيل الدخول من حساب محذوف، تظهر رسالة:
  ```
  تم حذف حسابك من النظام. يرجى التواصل مع الإدارة.
  ```

### 8. مؤشرات بصرية 🎨
#### في الجدول:
- Badge "موقوف" باللون البرتقالي للحسابات الموقوفة
- Badge "محذوف" باللون الأحمر للحسابات المحذوفة
- Badge "جديد!" باللون الأخضر للمستخدمين الجدد
- عرض سبب الإيقاف (أول 30 حرف) تحت اسم المستخدم الموقوف

#### في نافذة التفاصيل:
- عرض كامل سبب الإيقاف في قسم "حالة الحساب"
- عرض تاريخ الإيقاف
- أزرار ملونة حسب الإجراء:
  - برتقالي: إيقاف مؤقت
  - أخضر: تفعيل/توثيق
  - أحمر: حذف/رفض
  - بنفسجي: تغيير النوع

## كيفية الاستخدام

### إيقاف حساب:
1. افتح صفحة إدارة المستخدمين
2. اضغط على زر "إيقاف مؤقت" (أيقونة UserX)
3. أدخل سبب الإيقاف
4. اضغط "إيقاف الحساب"

### تفعيل حساب موقوف:
1. انتقل إلى تبويب "Inactive"
2. اضغط على زر "تفعيل الحساب" (أيقونة UserCheck)
3. راجع سبب الإيقاف السابق
4. اضغط "تفعيل الحساب"

### حذف حساب:
1. اضغط على زر "حذف الحساب" (أيقونة Trash2)
2. تأكد من الحذف
3. الحساب سينتقل إلى تبويب "Deleted"

### تغيير نوع الحساب:
1. اضغط على زر "تغيير النوع" (أيقونة UserCog)
2. اختر النوع الجديد من القائمة
3. اضغط "تغيير نوع الحساب"

### توثيق/رفض حساب:
1. للحسابات في حالة "في الانتظار"
2. اضغط على زر "توثيق" (CheckCircle2) للموافقة
3. أو اضغط على زر "رفض" (XCircle) للرفض

## البحث

يمكنك البحث عن المستخدمين باستخدام:
- الاسم
- البريد الإلكتروني
- رقم الهاتف (بأي تنسيق)

مثال:
```
0123456789
+20123456789
966512345678
```

## التصفية

الفلاتر المتاحة:
- نوع الحساب
- حالة التفعيل (Active/Inactive)
- حالة التوثيق (Verified/Pending/Rejected)
- اكتمال الملف الشخصي
- الدولة والمدينة
- طريقة التسجيل (مباشر/عبر منظمة)
- تاريخ التسجيل

## الأمان

- جميع الإجراءات محمية بـ `AccountTypeProtection` للإدارة فقط
- يتم تسجيل معرف المسؤول الذي قام بكل إجراء
- يتم تسجيل تاريخ ووقت كل إجراء
- لا يمكن حذف البيانات نهائياً من الواجهة

## ملاحظات مهمة

1. **الحذف منطقي**: الحذف لا يزيل البيانات فعلياً، بل يضع علامة `isDeleted: true`
2. **إمكانية التسجيل مرة أخرى**: المستخدم المحذوف يمكنه إنشاء حساب جديد بنفس البريد
3. **سبب الإيقاف إلزامي**: لا يمكن إيقاف حساب بدون ذكر السبب
4. **تغيير النوع حذر**: تغيير نوع الحساب قد يؤثر على البيانات المرتبطة
5. **التوثيق نهائي**: بعد التوثيق أو الرفض، لا يظهر الزرين مرة أخرى
