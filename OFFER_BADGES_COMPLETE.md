## ✨ **ما تم إضافته:**

### **شارة متحركة جميلة على كل باقة:**

```typescript
{appliedOffer && appliedOffer.applicablePlans?.includes(key) && (
  <div className="absolute -top-3 right-4 z-10">
    <motion.div
      initial={{ scale: 0, rotate: -180 }}
      animate={{ scale: 1, rotate: 0 }}
      transition={{  
        type: "spring",
        stiffness: 260,
        damping: 20
      }}
      className="px-3 py-1.5 bg-gradient-to-r from-green-500 to-emerald-600 rounded-full shadow-lg"
    >
      <motion.span
        animate={{ 
          scale: [1, 1.2, 1],
          rotate: [0, 10, -10, 0]
        }}
        transition={{
          duration: 2,
          repeat: Infinity
        }}
      >
        🎁
      </motion.span>
      خصم {appliedOffer.discountValue}%
    </motion.div>
  </div>
)}
```

## 🎨 **التصميم:**

### **الموقع:**
```
┌─────────────────────────┐
│          🎁 خصم 25%     │ ← أعلى يمين
├─────────────────────────┤
│   ⚡ باقة 3 شهور       │
│   ⭐ الأكثر شعبية       │ ← أعلى وسط
│                         │
│   $25/3 شهور           │
│   ✅ ميزة 1            │
│   ✅ ميزة 2            │
└─────────────────────────┘
```

### **التأثيرات (Framer Motion):**

1. **ظهور الشارة:**
   - Scale من 0 → 1
   - Rotate من -180° → 0°
   - نوع: Spring animation

2. **حركة الأيقونة 🎁:**
   - Scale: 1 → 1.2 → 1
   - Rotate: 0 → 10° → -10° → 0°
   - تتكرر كل ثانيتين

3. **تأثير اللمعان:**
   - opacity: 0 → 0.3 → 0
   - scale: 1 → 1.2 → 1
   - يتكرر باستمرار
