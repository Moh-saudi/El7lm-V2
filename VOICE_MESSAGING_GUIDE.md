## 🎯 كيفية الاستخدام:

### إرسال رسالة صوتية:
1. افتح محادثة
2. اترك حقل الرسالة **فارغاً**
3. اضغط على زر **🎤 الميكروفون**
4. **سجّل** رسالتك
5. اضغط **✓** لإيقاف التسجيل
6. اضغط **➤** للإرسال
7. انتظر رسالة "تم إرسال الرسالة الصوتية"

### تشغيل رسالة صوتية:
1. ابحث عن رسالة صوتية (🎤)
2. اضغط على زر **▶ Play**
3. الزر يتحول إلى **⏸ Pause**
4. اضغط مرة أخرى **للإيقاف**
5. الصوت يتوقف تلقائياً عند الانتهاء

## 📋 متطلبات التشغيل:

### 1. متغيرات البيئة (`.env.local`):
```env
# Cloudflare R2
CLOUDFLARE_R2_ENDPOINT=https://YOUR_ACCOUNT_ID.r2.cloudflarestorage.com
CLOUDFLARE_R2_ACCESS_KEY_ID=your_access_key
CLOUDFLARE_R2_SECRET_ACCESS_KEY=your_secret_key
CLOUDFLARE_R2_BUCKET_NAME=el7lmplatform
NEXT_PUBLIC_CLOUDFLARE_R2_PUBLIC_URL=https://pub-XXXXX.r2.dev
```

### 2. تثبيت التبعيات:
```bash
npm install @aws-sdk/client-s3
```

### 3. إعدادات Cloudflare R2:
- Bucket Name: `el7lmplatform`
- Public Access: **Enabled** (للمجلد `voice-messages/`)
- CORS: مفعل للنطاق الخاص بك
