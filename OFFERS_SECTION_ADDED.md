## ✅ **ما تم إضافته:**

### **1. قسم العروض المتاحة** 🎁

قسم جديد يظهر **في أعلى الصفحة** يعرض جميع العروض المتاحة بشكل جذاب:

```tsx
{availableOffers.length > 0 && (
  <div className="mb-8">
    <div className="p-6 bg-gradient-to-r from-green-50 to-emerald-50...">
      {/* العنوان */}
      <h3>العروض الترويجية المتاحة</h3>
      <p>{availableOffers.length} عروض متاحة الآن!</p>
      
      {/* قائمة العروض */}
      <div className="grid grid-cols-1 gap-4 md:grid-cols-2">
        {availableOffers.map((offer) => (
          <div>
            <h4>{offer.title}</h4>
            <p>{offer.description}</p>
            
            {/* شارات المعلومات */}
            <span>خصم {offer.discountValue}%</span>
            <span>حد أدنى {offer.minPlayers} لاعب</span>
          </div>
        ))}
      </div>
    </div>
  </div>
)}
```

### **الألوان:**

- ⚪ **بطاقات**: أبيض مع ظل
